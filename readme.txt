== Description ==

SideOffer is a pop-out content slider, designed to increase conversions by allowing you to present your users with a highly visible call to action. 

You could collect emails for your newsletter, offer a free download or make your contact form persistant. 

Features include: 

*	Easy and Unteractive Setup with real-time visual feedback
*	Anchor link triggering via inclusion the "sideoffer" class
*	Custom Background Graphics (PSD Source included) 

== Installation ==

1. Upload `sideoffer.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin through the SideOffer menu item
1. Optional: Include &lt;a href="javascript:void(0);" class="sideoffer"&gt;Links&lt;/a&gt; on your site to trigger the SideOffer
